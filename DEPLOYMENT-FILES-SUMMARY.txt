║                      QUICK START GUIDE                           ║
╚══════════════════════════════════════════════════════════════════╝

1️⃣  TEST LOCALLY
    ./test-docker-local.sh
    → Visit http://localhost:8000

2️⃣  DEPLOY TO AWS
    ./deploy-to-apprunner.sh
    → Builds & pushes to ECR

3️⃣  CREATE SERVICE
    → AWS Console → App Runner
    → Use ECR image from step 2
    → Add API keys as environment variables
    → Deploy!
